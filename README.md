# Склейка предлогов

Обычный WordPress-плагин для точечной русской типографики на historyalive.ru.

Плагин не меняет текст в базе данных. Он обрабатывает HTML только при выводе на фронтенде.

## Что делает

- Склеивает короткие русские служебные слова с последующим словом неразрывным пробелом в выбранном режиме.
- Склеивает исторические сокращения: `н. э.`, `до н. э.`
- Склеивает числа с частыми единицами: `5 км`, `480 году`, `24 часа`.
- Позволяет выбрать область применения: весь фронтенд или только одиночные записи и страницы.
- Позволяет отключить склейку коротких слов в заголовках, чтобы неразрывные пробелы не ухудшали переносы на узких экранах.

## Что не делает

- Не склеивает массово существительные с прилагательными.
- Не включает widow protection для последних слов абзаца.
- Не меняет содержимое `code`, `kbd`, `pre`, `samp`, `script`, `style`, `textarea`.
- Не обрабатывает REST API, feeds, AJAX и админку.
- В режиме «Только одиночные записи и страницы» не обрабатывает карточки, архивы и главную ленту.

## Настройки

Настройки доступны в админке WordPress: **Настройки → Склейка предлогов**.

- **Только одиночные записи и страницы**: обрабатываются основной текст, заголовок и комментарии на страницах записей и страниц. Карточки, архивы и главная лента не меняются.
- **Весь фронтенд**: типографика применяется к записям, страницам, карточкам, архивам, заголовкам, анонсам и комментариям на всём фронтенде.
- **Не склеивать короткие слова в заголовках**: оставляет обычные пробелы после коротких союзов, предлогов, частиц и местоимений в заголовках записей и внутри `h1`-`h6`. Исторические сокращения и числа с единицами в заголовках продолжают обрабатываться.
- **Склейка коротких слов: мягкий режим**: склеивает только односимвольные служебные слова `в`, `к`, `с`, `у`, `о`, `и`, `а` и только если следующее слово не длиннее заданного порога. Значение по умолчанию: `10` символов.
- **Склейка коротких слов: полный режим**: склеивает широкий список коротких союзов, предлогов, частиц и местоимений с последующим словом. Даёт более плотную типографику, но на узких экранах может усиливать рваность строк.
- **Склейка коротких слов: выключено**: не склеивает короткие слова, но оставляет активными исторические сокращения и числа с единицами.

## Проверка

```bash
php -l wp-content/plugins/wp-russian-preposition-glue/wp-russian-preposition-glue.php
```
